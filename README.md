
### 1. Установка зависимостей
Откройте терминал в папке проекта и выполните команду:
```bash
npm install
```

### 2. Запуск в режиме разработки
```bash
npm run dev
```
Откройте ссылку, которая появится в терминале (обычно http://localhost:5173).

### 3. Сборка для продакшена
```bash
npm run build
